# Linux Native Target Stdlib Matrix

Status: current for the `v0.4.0` Linux native target-family promotion work.

This matrix is a release-truth companion to
`docs/plans/2026-05/web-targets/linux_x86_x64_x32_full_support_plan.md` and
`docs/spec/runtime/runtime_abi.md`. It records current stdlib/runtime capability by target without
promoting `linux-x86` or `linux-x32` beyond available evidence. The same truth boundary is
machine-readable in `tetra targets --format=json`: Linux native entries include `runtime_status`,
`stdlib_status`, `ffi_status`, `runner_probe_command`, `release_gate`, and `evidence_artifacts`,
with x86/x32 remaining `partial_build_only` until the listed release reports prove more. They also
include the syscall pack (`syscall_instruction`, `syscall_numbering`, `syscall_arg_registers`,
`syscall_error_range`) so the i386, x86_64, and x32 syscall boundaries are validated as target
facts.
