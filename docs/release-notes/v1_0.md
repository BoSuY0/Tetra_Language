# Tetra v1.0 Release Notes

Status: final release candidate snapshot for `v1.0.0`.

This release records the verified local compiler/tooling profile for Tetra
`v1.0.0`.

## Verified In This Snapshot

- Docs manifest regenerated and validated.
- Docs verification and doctests pass.
- Build-only smoke passes for all mandatory targets: `linux-x64`, `macos-x64`,
  `windows-x64`, `wasm32-wasi`, `wasm32-web`.
- Native host smoke (`linux-x64`) runs.
- WASI smoke runs with a scripted fallback runner (`node-wasi`) and emits a
  validated smoke report.
- Web UI smoke runs against `examples/projects/dogfood_web_ui/src/main.tetra`
  with browser automation and a schema-validated report.
- API diff baseline + candidate diff are generated and enforced in
  `no-change` mode.
- Reproducible build proof passes for one native target (`linux-x64`) and one
  WASM target (`wasm32-wasi`).
- `./tetra version` and `./t version` report `v1.0.0` after bootstrap.
