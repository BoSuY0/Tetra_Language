# Tetra v1.0 Release Notes

Status: bounded `v1.0.0` release notes. The current supported surface is
`docs/spec/current_supported_surface.md`, and the normative scope contract is
`docs/spec/v1_scope.md`.

This release promotes the local compiler/toolchain profile with runner-backed
WASM runtime evidence, browser-backed Web UI smoke evidence, and Linux-x64
desktop GUI runtime evidence. It preserves the v0.4 Linux-x64 release truth as
historical evidence and does not claim EcoNet production, hosted TetraHub,
Windows/macOS GUI runtime, or full first-class language guarantees.

The final release decision is made by `scripts/release/v1_0/gate.sh`; these
notes are release truth only when that gate passes on the same commit, report
directory, security review, artifact hashes, and release-state audit.

Previous release notes live in `docs/release-notes/v0_4_0.md`.
