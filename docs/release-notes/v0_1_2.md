# Tetra v0.1.2 Release Notes

Status: historical release snapshot. It is not current release truth; see
`docs/spec/current_supported_surface.md` for the current public profile.

This release records the verified local compiler/tooling profile for Tetra
`v0.1.2`. It is a stabilization release, not the final `v1.0.0` language
label.

## Verified In This Snapshot

- Docs manifest regenerated and validated.
- Docs verification and doctests pass.
- Build-only smoke passes for all current release targets:
  `linux-x64`, `macos-x64`, `windows-x64`, `wasm32-wasi`, and `wasm32-web`.
- Native host smoke (`linux-x64`) runs on Linux.
- WASI smoke runs with the scripted fallback runner and emits a validated
  report.
- Web UI smoke runs against `examples/projects/dogfood_web_ui/src/main.tetra`
  with browser automation and a schema-validated report.
- API diff baseline plus candidate diff are generated and enforced in
  `no-change` mode.
- Reproducible build proof passes for one native target (`linux-x64`) and one
  WASM target (`wasm32-wasi`).
- `./tetra version` and `./t version` report `v0.1.2` after bootstrap.
