# Tetra v0.3.0 Release Notes

Status: historical release notes for the `v0.3.0` profile.

## Highlights

- Promotes positional enum payload constructors and payload bindings for
  match/catch/if-let with exhaustive unguarded enum match/catch diagnostics.
- Promotes static protocol-bound generic validation during monomorphization,
  including same-module and cross-module conformance/visibility diagnostics.
- Adds a dedicated `v0.3.0` release gate and checklist.
- Keeps Callable Level 1, lifetime SSA, WASM runtime execution, distributed Eco,
  and full `v1.0.0` guarantees outside the stable support claim.
