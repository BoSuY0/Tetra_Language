# P25.0 Real Local Benchmark Execution V1 Plan

**Goal:** Produce Tier 1 local benchmark evidence for the P20 matrix without
promoting any fastest-language, official benchmark, cross-machine,
TechEmpower, or production claim.

**Context:** P20.0 already hardens the benchmark matrix contract through
`tools/cmd/truth-bench-harness`, but the checked report is a dry run with
`ran=false`. P25.0 must attempt real local execution for Tetra, C `clang -O3`,
C++ `clang++ -O3`, and Rust `rustc -C opt-level=3`, record measured local
evidence where available, and classify every category honestly.

## Task 1: Schema And Validator

- **Goal:** Define and validate the P25.0 Tier 1 report schema.
- **Files:** Add a validator under `tools/cmd/validate-local-benchmark-tier1/`
  or extend a nearby benchmark validator if a better existing integration point
  appears during implementation.
- **Approach:** Require all 17 P20 categories, all four languages, command
  evidence, raw artifacts, measured attempts, classification values, non-claims,
  and Tetra proof/bounds/allocation/perf/backend/optimizer metadata.
- **Verification:** Focused `go test` for the validator package, then
  `go run ./tools/cmd/validate-local-benchmark-tier1 --report
  reports/local-benchmark-tier1-v1/report.json`.
- **Done when:** A malformed report without matrix rows, Tetra metadata, raw
  artifacts, or non-claim policy is rejected.

## Task 2: Real Local Runner

- **Goal:** Generate and run deterministic local P20 workloads.
- **Files:** Add a runner under `tools/cmd/local-benchmark-tier1/` unless an
  existing harness extension proves smaller.
- **Approach:** Write source files and artifacts under
  `reports/local-benchmark-tier1-v1/`, compile C/C++/Rust with the required
  optimization commands, build Tetra with `go run ./cli/cmd/tetra build
  --target linux-x64 --explain`, run each available binary, capture stdout,
  stderr, exit code, elapsed time, compile time, binary size, and source paths.
  Unsupported PostgreSQL live execution is allowed to become
  `blocked by missing feature` or `invalid/inconclusive` with exact evidence.
- **Verification:** Run the tool locally with a persistent Go build cache under
  `.cache/go-build-p25-tier1`.
- **Done when:** `report.json` records real command attempts and raw artifacts
  for every row.

## Task 3: Tetra Metadata Extraction

- **Goal:** Attach Tetra proof, bounds, allocation, perf blocker, backend path,
  and optimizer validation metadata to every Tetra row.
- **Files:** Runner code and report artifacts produced next to each Tetra
  binary.
- **Approach:** Parse `.proof.json`, `.bounds.json`, `.alloc.json`,
  `.perf.json`, and `.backend.json`; summarize backend as `register`, `stack`,
  or `fallback`; link optimizer validation to the current P23 translation
  validation evidence if no per-benchmark optimizer report exists.
- **Verification:** Validator rejects any Tetra row without these metadata
  fields or artifact existence.
- **Done when:** Every Tetra category has artifact paths and parsed metadata in
  `report.json`.

## Task 4: Classification And Docs

- **Goal:** Classify every category and write user-facing evidence summaries.
- **Files:** `reports/local-benchmark-tier1-v1/report.json`,
  `reports/local-benchmark-tier1-v1/summary.md`, and
  `docs/audits/local-benchmark-tier1-v1.md`.
- **Approach:** Use explicit local-only classification rules. Prefer blocker
  classifications when Tetra cannot build/run, uses fallback backend, carries
  heap/bounds blockers, or actor/runtime limits apply; otherwise compare local
  measured medians with a conservative comparable threshold.
- **Verification:** Validate the schema and run docs/manifest checks.
- **Done when:** Summary and audit cite report paths, commands, classifications,
  and non-claims without overstatement.

## Task 5: Required Gates

- **Goal:** Prove the artifact set against the required repo gates.
- **Files:** No planned source edits beyond the runner/validator and generated
  P25 report/audit artifacts.
- **Approach:** Run:
  `go run ./tools/cmd/validate-local-benchmark-tier1 --report reports/local-benchmark-tier1-v1/report.json`,
  `go run ./tools/cmd/verify-docs --manifest docs/generated/manifest.json`,
  `go run ./tools/cmd/validate-manifest --manifest docs/generated/manifest.json`,
  `git diff --check`, `graphify update .`, and
  `go test ./compiler/... ./cli/... ./tools/... -count=1`.
- **Verification:** Read exit status and output for each command.
- **Done when:** Passing commands are recorded, or any failure is reported as a
  real blocker with exact command evidence.
