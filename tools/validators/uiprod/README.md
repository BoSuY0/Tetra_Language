# tools/validators/uiprod

Validator package for executable UI Production Runtime evidence.

This boundary owns the `tetra.ui.desktop-runtime.v1` report contract. A passing
report must show real Linux-x64 desktop UI runtime process evidence, widget
hierarchy, state binding, input/list/button events, async command completion,
timer tick and redraw operations, stable negative diagnostics, dogfood examples,
native shell integration, widget stress, and completion-audit rows.
