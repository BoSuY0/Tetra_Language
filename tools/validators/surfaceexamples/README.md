# Surface Example Suite Validator

`tools/validators/surfaceexamples` validates
`tetra.surface.example-suite-report.v1` evidence for
`surface-production-example-suite-v1`.

The package owns realistic app-shape coverage across the scoped Surface
production examples. It checks source, target, Block/Morph, accessibility,
performance, and nonclaim evidence so example-suite rows cannot stand in for
unsupported platform parity.
