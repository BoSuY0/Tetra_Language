# Surface Electron Comparison Validator

`tools/validators/surfaceelectron` validates
`tetra.surface.electron-comparison-report.v1` evidence for
`surface-electron-comparison-method-v1`.

The package owns scoped Surface-vs-Electron positioning and comparison method
checks. It accepts only competitive-in-supported-scope claims and rejects
official benchmark, faster-than-Electron, full Electron parity, and React/CSS
compatibility overclaims.
